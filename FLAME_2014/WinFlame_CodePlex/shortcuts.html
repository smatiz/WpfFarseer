The scriptercontrolForm comes with some shorcuts with which you can control the application.

When you want compile the text of the current cell

SHIFT + ENTER execute script

When you need help

CTRL + SPACE
 show intellisense : autocomplete or inline help depending on context

When the cursor is in one cell you can change its current language, this will change the syntax highlight, the autocomplete and the compilator

CTRL + ALT + SHIFT + P change language of the current cell to IronPython
CTRL + ALT + SHIFT + R change language of the current cell to IronRuby ( default, best support at this time )
CTRL + ALT + SHIFT + C change language of the current cell to C# 
CTRL + ALT + SHIFT + S change language of the current cell to Powershell
CTRL + ALT + SHIFT + J change language of the current cell to IronJS ( poor support )

You can also create, move, delete, duplicate cells

ALT + N
 new cell ( below the current )
ALT + D
 duplicate current cell ( below )
ALT + L
 copy the cell above into the current
ALT + R
 delete current cell
ALT + DOWN/UP
 move down/up the current cell
ALT + PGDOWN/PGUP move cursor below/above the current cell

The scriptercontrolForm autosaves code when you close it but you can also force to save

CTRL + S
 show intellisense : autocomplete or inline help depending on context

ScintillaNET controls comes with its own shortcuts, almost any of them are still active (if they are still useful, ex: print has been removed).
